# 不实现课前测

学习循环不包含独立的"课前测"步骤。学生进入 KP 后直接进入苏格拉底式讲解；初始摸底由对话开篇的提问承担。

## Considered Options

- **每 KP 自动课前测**：增加独立的出题、判分、决策模块。工作量与教学价值不成比例：苏格拉底对话本身就在持续测，独立课前测会重复劳动。
- **条件性课前测（仅在已通过 KP 复习时出题）**：属于间隔重复方向的产品扩展，不在毕设范围内。

## Consequences

- 苏格拉底对话的开篇 prompt 需要承担"摸底"职责（开场要主动问"你对 X 已经知道什么"），这是 prompt 工程的硬要求。
- 复习已通过 KP 的入口暂不实现，相关需求（间隔重复、自适应复习）记为未来扩展方向。
- `毕设.md` 中提到的"课前测"概念在最终系统中不存在，文档需要一致更新。
