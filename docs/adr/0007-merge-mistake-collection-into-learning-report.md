# 错题集与学习报告页合并

不实现独立的"错题集"页。错题与薄弱知识点的展示统一在"学习报告页"中处理。

## Considered Options

- **独立错题集页**（`毕设.md` 原始构想）：错题逐题列出，支持单题复习作答。
- **学习报告页 + 错题区块**（当前选择）：错题作为薄弱知识点的具象表现，挂在薄弱点详情下。

## Consequences

- 学习报告页的信息密度增加，需要清晰分区（薄弱知识点列表 / 历史错题 / 进度统计）。
- "错题"作为概念在系统中不再独立；存储时仍记录每一道答错的题（用于后续作业的复习题混入，参见 ADR-0005），但展示时归在薄弱点的下钻视图里。
- `毕设.md` 中提到的"错题集"页面在最终系统中不存在，文档需要一致更新。
