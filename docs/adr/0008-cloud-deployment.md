# 部署形态：云端 Web 服务

系统部署到云端 Web 服务（前端如 Vercel、后端如 Render / Railway），评委可远程访问。本地开发环境保留作演示备份。

## Considered Options

- **本地 Web（仅你电脑跑）**：演示最稳，但缺产品形态展示，论文中无法引用"已部署版本"。
- **桌面应用（Tauri / Electron）**：打包工作量大、跨 OS 测试成本高。

## Consequences

- 部署调试工作量约 1 天（CORS、HTTPS、环境变量、数据库远程连接）。
- 演示日有云服务故障 / 网络故障风险，**必须保留本地备份方案**：本地仍能完整运行。
- 论文中可以写"已部署可远程访问"，提供答辩 +1 的产品感。
- API key 等敏感信息要正确放进环境变量，不能 commit。
