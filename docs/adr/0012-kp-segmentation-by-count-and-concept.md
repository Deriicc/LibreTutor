# KnowledgePoint 切分量化：按数量目标 + 单一概念约束

章节树生成的 prompt 中，KP 切分采用双重约束：

1. **数量目标**：每个 Section 切 3-7 个 KP（实际数字可在 prompt 调试中调整）。
2. **概念约束**：每个 KP 聚焦一个核心概念，不允许多概念混合。

不使用"按字数"或"按 5-15 分钟时长"的显式约束。

## Considered Options

- **按字数约束（"每个 KP 200-500 字"）**：会让 LLM 机械切分，破坏语义边界。
- **按时长约束（"每个 KP 5-15 分钟"）**：LLM 对学生阅读速度无感知，不可量化。
- **不显式约束，让 LLM 自行判断**：结果颗粒度极不稳定，同一 PDF 多次生成差异大。

## Consequences

- 80 页 PDF 估算产出 50-120 个 KP，量级合理。
- 实际生成数量超出预期时，调 prompt 中的范围数字（如改成 2-5 个/Section）。
- 切分质量依赖 LLM 对"单一核心概念"的理解，prompt 中需要给出 1-2 个示例（few-shot）。
- 生成结果的稳定性需要在 prompt 调试期间多次采样验证。
